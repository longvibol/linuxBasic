Step by Step 

>> sudo apt-get update : update 

>> sudo apt install samba : install samba

>>whereis samba : check it is install in our directory or not 

>> mkdir /home/vibol/sambashare/ :make directory 

>> sudo nano /etc/samba/smb.conf : edit our directory 

[sambashare]
        comment = Samba on Ubuntu
        path = /home/vibol/sambashare
        read only = no
        browsable = yes

==> Ctrl + X --> Y to save file conf 


>> sudo smbpasswd -a vibol :create user for other login:

vibol
admin

>> ifconfig : to check our ip address

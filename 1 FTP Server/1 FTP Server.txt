ftp: Login failed

Result we can not log in => so we can edit in the vsftpd.conf file 

4. Edit file 

>> sudo nano /etc/vsftpd.conf


What we edit : 
- listen=YES
- listen_ipv6=NO
- anonymous_enable=YES
- local_enable=YES

>> ctlt + X to save and exit 

5. Restart service vsftpd

>> sudo systemctl restart vsftpd
>> systemctl status vsftpd.service


6. Try to login again
